# v1.0.5

- **发布日期**：2026-07-10

## 新增功能

### Skill 市场与在线安装

- 新增 **Skill 市场**页面，支持浏览、搜索和查看 Skill 详情（README、安装指引、语法高亮）。
- 支持从 Skill 市场**在线安装 Skill 到运行中的 Agent 实例**，无需重启即可生效。
- Skill 动态选择挂载：创建实例时支持从已安装的 Skill 列表中选择并挂载。
- Skill 市场 UI 对齐计算巢：卡片样式、详情页布局、SkillDisplayName 独立字段等全面对齐。

### OOS Checkpoint 备份恢复

- 新增基于 OOS（运维编排服务）的 **Checkpoint 自动备份与恢复**能力。
- 支持动态发现 Sandbox 目标，自动执行备份任务。
- OOS 备份执行权限收敛，仅授权必要操作。

### LiteLLM 增强

- LiteLLM 域名自适应：根据部署环境自动选择 LiteLLM 网关域名。
- 网关监控页面新增跳转 LiteLLM 详情页入口。

### 其他改进

- Skill 卡片头像字母和颜色改为基于 displayName 生成（对齐计算巢）。
- Harness 隔离环境生命周期流水线（CI 改进）。

## 升级步骤

请参考 [Release Notes 通用升级方式](./README.md#通用升级方式)：在**计算巢控制台**对服务实例执行**升级**操作即可，数据库迁移会随升级流程自动完成。

## 注意事项

- Skill 在线安装依赖 Agent 实例处于运行状态，离线实例需先启动后安装。
- OOS 备份恢复功能需确保目标集群已部署 OOS 相关组件。
