## 通用升级方式

所有版本的升级均通过**计算巢控制台**完成：

1. 登录 [阿里云计算巢控制台](https://computenest.console.aliyun.com/service/instance/cn-hangzhou)。
2. 在「服务实例管理」中找到目标 Agent Manager 服务实例。
3. 点击实例进入详情页，右上角选择「升级服务实例」操作。
4. 在升级向导中选择目标版本，确认参数后提交。
5. 等待 ROS 资源栈完成更新（升级过程中会自动执行数据库增量迁移，无需人工介入）。

> 提示：升级期间会进行滚动更新，建议在业务低峰期执行。

## 新增版本说明文件

每发布一个新版本，请按照如下规范在本目录新增一个 `vX.Y.Z.md` 文件：

- **新增功能**：从用户视角描述本版本带来的能力变化，无需逐条罗列 commit。
- **问题修复**（可选）：列出对用户感知明显的修复项。
- **升级步骤**：统一引用本文件中的「通用升级方式」。
- **注意事项**（可选）：版本特有的兼容性、配置变更或操作建议。
