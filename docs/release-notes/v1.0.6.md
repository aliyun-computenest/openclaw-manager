# v1.0.6

- **发布日期**：2026-07-17

## 新增功能

### AI 应用可观测性（APM）

- 新增 **AI 应用监控**能力，支持三类 Agent（OpenClaw / Hermes / QwenPaw）接入阿里云 ARMS APM。
- **新建即用**：新建 Agent 实例后发起对话，监控数据自动上报，无需手动安装探针或配置参数。
- **应用监控面板**：管理员可在「应用监控」页面查看每个实例的调用链路、对话分析、Token 消耗、模型调用耗时等指标（通过 CMS iframe 免登嵌入）。
- **采集开关**：管理员可在 Agent Type 配置页一键开启/关闭监控采集，切换后秒级生效，无需重建实例。
- **自动参数初始化**：服务首次部署时自动从 CMS API 获取 ARMS 接入参数并写入配置，管理员无需手动填写。

### 应用监控诊断

- 应用监控页面提供实例级状态诊断：自动检测参数配置、采集开关、探针安装、实体注册等环节，并给出针对性引导提示。
- 采集关闭时展示提示页面，支持一键跳转至 Agent Type 配置页开启采集。

## 升级步骤

请参考 [Release Notes 通用升级方式](./README.md#通用升级方式)：在**计算巢控制台**对服务实例执行**升级**操作即可，数据库迁移会随升级流程自动完成。

## 注意事项

- **xtrace 写入通道**：升级后需在阿里云云监控 2.0 控制台确认对应工作空间的 trace 写入通道已开启，否则监控数据无法上报。
- **老实例接入**：已运行的老实例需通过「实例升级」功能更新镜像后方可接入应用监控；新建实例无需任何额外操作。
- **采集开关默认状态**：升级后内置 Agent Type 的采集开关默认为**关闭**，管理员需手动开启后新建实例才会上报监控数据。
- **QwenPaw 已知限制**：QwenPaw 的链路追踪中「输入/输出」内容展示依赖 LoongSuite 插件版本，当前 0.7.0 版本存在 ContextVar 兼容性问题，待 LoongSuite 团队后续修复。
