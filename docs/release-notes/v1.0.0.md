# v1.0.0

- **发布日期**：2025-04-14

## 版本概述

v1.0.0 是 Agent Manager 平台的首个正式发布版本，提供了 OpenClaw 智能体在云端的一站式管理能力。核心能力包括：

- **服务实例管理**：在阿里云计算巢上一键部署 Agent Manager，支持开箱即用的实例创建、查看、销毁等生命周期管理。
- **用户与权限管理**：内置基于 Supabase 的用户认证体系，支持普通用户与管理员两类角色，管理员可统一管理用户与配额。
- **模型与渠道配置**：支持配置 OpenAI 兼容的大模型与 AI 网关渠道，统一管理 API Key 与额度。
- **OpenClaw Agent 运行环境**：基于 E2B 沙箱为每个用户提供隔离的智能体运行容器。
- **基础运维能力**：提供平台监控、用量查询与基础的健康检查能力。

## 升级步骤

v1.0.0 为初始版本，无需升级。新部署请参考[计算巢服务页](https://computenest.console.aliyun.com/) 选择 Agent Manager 服务进行部署。
